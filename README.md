# 🎓 University Support Chatbot

An intelligent chatbot built using Streamlit that helps students get instant answers about exam schedules, syllabus, and contact details using a structured JSON database.

---

## 🚀 Features

* 💬 ChatGPT-style chatbot interface
* ⚡ Quick questions (Exam Schedule, Syllabus, Contacts)
* 📂 JSON-based knowledge system
* 🧠 AI-powered responses
* 🔄 Chat history with clear option
* 🎯 Accurate answers based only on database

---

## 🛠️ Tech Stack

* **Frontend + Backend:** Python (Streamlit)
* **AI Model:** Large Language Model (LLM)
* **Database:** JSON
* **Libraries:**

  * streamlit
  * json

---

## 🧠 How It Works

1. User enters a question in the chat interface
2. The system processes the query along with the JSON database
3. The AI model analyzes the data
4. A relevant response is generated and displayed

## ⚙️ Installation

### 1. Clone Repository

```
git clone https://github.com/your-username/university-chatbot.git
cd university-chatbot
```

---

### 2. Install Dependencies

```
pip install streamlit
```

---

### 3. Run the App

```
streamlit run chatbot.py
```
